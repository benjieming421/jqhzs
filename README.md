## 使用jQuery的环境

1. jQuery v3.4.1

## 使用jqhzs

1. 页面需要引用jquery文件
2. 页面需要引用jqhzs文件

```javascript
    <script src="./jquery.js"></script>
    <script src="./jqhzs.js"></script>
```

   以上引用注意顺序

```
$_ 为开头

如 `$_.alerton()`
```
